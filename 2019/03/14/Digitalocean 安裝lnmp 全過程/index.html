Digitalocean 安裝 lnmp 全過程

發表於 2019-03-14 | 分類於 LINUX ， 雲 |
  1. 在 Digitalocean 新增一個 linux 系統主機，系統我選擇了 CentOS 7.5 x64
  2. 配置根據你需要安裝的軟件決定，因為我要裝PHP7.2 所以我選的是 10 美元 1 個月 2GB 記億體 CPU 1 核心 配置

安裝MySQL 5.6或5.7及MariaDB 10必須1GB以上記億體，更高版本至少要2GB記億體！ 安裝PHP 7及以上版本必須1GB以上記億體

  1. Digitalocean 會把 SSH 的帳號密碼資料發到你的郵件上，我是用 xshell 連接
  2. 連接後防止斷線後看不到安裝過程 首先安裝 screen 並開一個新視窗

CentOS系統可以執行：yum install screen ; Debian / Ubuntu系統執行：apt-get install
screen

  1. 然後執行：screen -S lnmp 建立新視窗
    如果網絡斷線，可以重新連接 SSH，再執行screen -r lnmp就會看到你的 lnmp 安裝進度。
    有時候會提示狀態為 Attached，可以執行：screen -D -r lnmp 進行恢復。
  2. 使用無人值守命令生成工具 根據你的需求生成所需命令
    https://lnmp.org/auto.html 無人值守命令生成工具地址

7.我選的是 MySQL 5.6 和 PHP 7.2
命令如下：

wget http://soft.vpser.net/lnmp/lnmp1.5.tar.gz -cO lnmp1.5.tar.gz && tar zxf lnmp1.5.tar.gz && cd lnmp1.5 && LNMP_Auto="y" DBSelect="3" DB_Root_Password="TEST123" InstallInnodb="y" PHPSelect="8" SelectMalloc="1" ./install.sh lnmp

注意 請自行根據你的需求生成所需命令 我的只是測試用

  1. 等待安裝完成 瀏覧自己主機的ip即可看到 lnmp測試頁面。

  2. 大功告成
